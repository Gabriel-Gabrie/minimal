🛠️ Budget App Helper Functions Test Suite

This page tests all 7 budget helper functions implemented in Phase 4.

The tests verify correct behavior and localStorage persistence (saveData() calls).

Setup

Test 1: getActiveSections(monthKey)

Should return active sections object for a month, or empty object if month doesn't exist.

Test 2: addSectionToBudget(monthKey, sectionName)

Should add section to month's activeSections and sectionOrder, initialize budgets structure.

Test 3: addCategoryToBudget(monthKey, sectionName, categoryName)

Should add category to section's categories array and initialize budgets structure.

Test 4: isSectionActiveInAnyBudget(sectionName)

Should return true if section is active in at least one month, false otherwise.

Test 5: isCategoryActiveInAnyBudget(sectionName, categoryName)

Should return true if category is active in section in at least one month, false otherwise.

Test 6: removeCategoryFromBudget(monthKey, sectionName, categoryName)

Should remove category from section's categories array and budgets structure.

Test 7: removeSectionFromBudget(monthKey, sectionName)

Should remove section from activeSections, sectionOrder, and budgets.
